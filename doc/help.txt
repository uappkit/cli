
uapp 通用命令🔥

  - 初始化或更新 uappsdk
    uapp sdk init

  - 生成的签名文件 app.keystore
    uapp keygen

  - 显示APP开发中会用到的配置信息
    uapp info [jwt|key] (不指定 jwt|key 会自动判断)

  - 新建一个基于 uniapp-cli 的工程，默认使用 vite vue3
    uapp new ${projectName}

    // 用 alpha 模版
    uapp new ${projectName} --alpha

    // 使用旧的 vue2 模版
    uapp new ${projectName} --vue2

需在工程模板下运行命令☘️

  - 核心命令: 读取 manifest.json 配置, 并更新 android 或 ios 工程
    uapp manifest sync ${webapp}/src/manifest.json
    uapp manifest info (仅查看信息, 不执行 sync)

  - 复制 HBuilderX 生成的本地打包APP资源
    uapp prepare

  - 发布离线打包基座到 HBuilderX 下
    uapp run build:dev --sync (uapp publish debug 将弃用)

  - 编译开发包，不发布
    uapp run build:dev

  - 编译 release 包
    // 注意: 因为iOS发布，需要走苹果的发布流程，所以不支持iOS，iOS发布直接用 xcode
    uapp run build

其他命令:

    uapp -v 或 --version (查看 uapp 版本)
    uapp help 或 -h 或 --help (查看帮助)

示例:

    uapp keygen
    uapp info jwt
    uapp -h
    uapp --version
