
uapp 通用命令🔥

  - 初始化或更新 uappsdk
    uapp sdk init

  - 新建指定平台工程
    uapp add android  // 当前目录下创建 android 工程
    uapp add ios      // 当前目录下创建 ios 工程

  - 显示APP开发中会用到的配置信息
    uapp info [jwt|key] (不指定 jwt|key 会自动判断)

  - 生成 android 的签名文件 app.keystore
    uapp keygen

  - 新建一个基于 uniapp-cli 的工程，默认使用 vite vue3
    uapp new ${projectName}
    uapp new ${projectName} --alpha --typescript
    uapp new ${projectName} --vue2 (旧的 vue2 模版)

需在工程模板下运行命令☘️

  - 核心命令: 读取 manifest.json 配置, 并更新 android 或 ios 工程
    uapp manifest path/to/manifest.json

  - 更新打包用到的数据，包括版本信息, dcloudkey, App图标, 本地打包资源等
    uapp prepare
    uapp prepare --no-webapp (不编译 webapp)

  - 编译并发布自定义基座到 HBuilderX 下
    uapp run build:dev
    uapp run build:dev --no-copy  (不发布)

  - 编译 release 包
    // iOS 平台请直接使用 Xcode 发布
    uapp run build

  - 运行自定义打包发布流程，配置见 README.md
    uapp run custom

其他命令:

    uapp -v 或 --version (查看 uapp 版本)
    uapp help 或 -h 或 --help (查看帮助)
